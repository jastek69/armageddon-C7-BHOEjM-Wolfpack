Python Tooling Connection:
  The SEIR_Foundations/LAB2/python/ folder contains tools directly
  relevant to Be A Man verification:
  - malgus_cloudfront_cache_probe.py: Runs repeated requests and prints
    x-cache/age/cache-control per round. Directly produces the ManA
    Miss→Hit→Miss evidence and D.1/D.2 proof programmatically.
  - malgus_origin_cloak_tester.py: Tests CF URL vs ALB URL, prints
    PASS/FAIL for origin cloaking (2A requirement).
  - run_all_gates.sh: Full infrastructure gate that checks CF, WAF,
    R53, ACM, TLS, SG, logging. Outputs gate_result.json + badge.txt.
    Running this gives an objective infrastructure grade.
